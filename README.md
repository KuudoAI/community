# Kuudo Community

The front door for feedback, questions, and troubleshooting — for everyone running Amazon with AI agents through Kuudo.

**Where to go:**

- 🐛 **Something broke?** [Report a bug](https://github.com/KuudoAI/community/issues/new?template=bug_report.yml)
- 💡 **Want something Kuudo doesn't do?** [Request a feature](https://github.com/KuudoAI/community/issues/new?template=feature_request.yml)
- 💬 **Question or troubleshooting?** [Start a discussion](https://github.com/orgs/KuudoAI/discussions)
- 🗺️ **Wondering what's next?** [The roadmap](https://github.com/orgs/KuudoAI/projects) — 👍 the issues you want prioritized
- 📚 **Learning?** [Field guides](https://www.kuudo.com/guides/) and [docs](https://www.kuudo.com/docs/) live on kuudo.com

The code lives in [`amazon_ads_mcp`](https://github.com/KuudoAI/amazon_ads_mcp) — our open-source MCP server for the Amazon Ads API.
